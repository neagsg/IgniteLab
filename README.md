## :fire: Installation

```bash
# Open a terminal, copy and paste the code below
$ https://github.com/neagsg/IgniteLab

# Access the project folder
$ cd IgniteLab

# Install the dependencies
$ yarn install
or
$ npm install

# Run the application
$ yarn dev
or 
$ npm run dev

# Access http://localhost:3000 in your browser 
```
